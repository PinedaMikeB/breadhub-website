# BreadHub Website

Firebase-powered e-commerce website for BreadHub bakery in Taytay, Rizal.

## Files

- `index.html` - Home page with SEO content and category cards
- `products.html` - Product catalog with cart and checkout
- `admin.html` - Admin panel for managing products
- `sitemap.xml` - SEO sitemap
- `robots.txt` - Search engine configuration
- `.htaccess` - Server optimization rules
